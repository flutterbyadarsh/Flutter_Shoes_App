A premium, high-performance E-commerce UI for a footwear store. Built with **Flutter**, this app focuses on smooth animations, clean typography, and an intuitive shopping experience.

---

## ✨ Key Features

* **👟 Modern Product Gallery:** Beautifully rendered shoe cards with category filtering.
* **🛒 Seamless Cart Management:** Add, remove, and update items with real-time feedback.
* **🔍 Interactive Product Details:** High-resolution imagery, size selection, and description toggles.
* **📱 Fully Responsive:** pixel-perfect design for both iOS and Android.
* **🎨 Custom UI Components:** Hand-crafted widgets for a unique "boutique" feel.
